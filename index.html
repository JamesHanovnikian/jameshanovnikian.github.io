Projects

Blade Spots

A web application where users can share/search for skate spots in their area and filter based on different categories and location. Features a map page where skate spots are pinned utilizing the Mapox API.

The idea for an app like this came to me almost a decade ago as an avid skater and I'm happy I was able to bring it to fruition. It solved a real problem I had of having to spend time seeking out street skate spots through trial and error. This app would also allow users to write a description on a skate spot page which would give skaters an idea of what they are getting into before investing the time to get there.

I used Ruby on Rails to build out the back-end infrastructure and Vue.js on the front end, along with HTML, CSS and Bootstrap 5.
