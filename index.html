Hi, I'm James.


Over the past few years I've had positions in a diverse group of industries including Transportation, Customer Service, and Education. A through line in these experiences has been a desire to problem solve and make an impact as a part of a team.

Working in transportation, I honed my skills in attention to detail and how to prioritize tasks in a time sensitive environment. Recently, I have worked in education as a teacher's assistant. I have always loved the learning process and seeing the progress students make is incredibly rewarding.

I've always been attracted to design and I took a few courses in web & graphic design in high school and college. I didn't revisit this interest until a few years later when I began self studying web development using online resources. The hobby turned into a full-time pursuit when I enrolled in Actualize Coding Bootcamp. Through Actualize's program, I've gained experience in Ruby, Rails, JavaScript, Vue.js, APIs, OOP, HTML, CSS, and Git. I'm eager to continue the learning process and I'm driven to use the culimination of my skills to make technology improve the lives of people.

Portfolio

Blade Spots

A web application where users can share/search for skate spots in their area and filter based on different categories and location. Features a map page where skate spots are pinned utilizing the Mapox API. I had an idea for an app like this almost a decade ago as an avid skater and I had a lot of fun bringing it to life.

I used Ruby on Rails and PostgreSQL to build out the back-end infrastructure and Vue.js on the front end, along with HTML, CSS and Bootstrap 5.

JavaScript Framework Watcher

An interactive dashboard that displays information about popular JavaScript frameworks like Vue.js, Svelte, React, & Ember. Data about the stars, watchers, and forks on each repository is taken from the Github API and displayed using chart.js. Built using JavaScript.
